# EcoTrack

![EcoTrack Logo](https://via.placeholder.com/150)

## Overview
EcoTrack is a sleek, user-friendly application designed to help individuals monitor and reduce their carbon footprint via daily activity logging and actionable insights. Stay green and make sustainable choices with ease.

## Features
- Log daily activities impacting your carbon footprint
- Visualize your carbon reduction progress over time
- Personalized tips to lower emissions
- Weekly and monthly reports with actionable suggestions
- Lightweight and intuitive interface

## Tech Stack
- **Frontend:** React.js
- **Backend:** Node.js with Express
- **Database:** MongoDB
- **Authentication:** JWT
- **Deployment:** Docker, Heroku

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/allie2023cup/EcoTrack.git
   cd EcoTrack
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Setup environment variables:
   Create a `.env` file with the following:
   ```
   MONGO_URI=your_mongodb_connection_string
   JWT_SECRET=your_jwt_secret
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```

## Usage
- Register an account or login using existing credentials.
- Start logging your daily activities.
- View your dashboard to monitor progress.
- Check tips and reports to stay motivated.
